# 音乐 app

名称： 音乐 APP
技术栈：vue(v2.6.6) + vue-router(v3.0.2) + vuex(v3.1.0) + vue-cli(v3.5.1) + sass + axios

# 项目启动

1. 克隆地址到本地

```js
git clone https://github.com/hucongcong/vue-music.git
```

2. 安装依赖

先进入到 vue-music 项目目录

```bash
yarn
# 如果没有yarn
npm install
```

3. 启动项目

```bash
yarn serve
# 如果没有yarn
npm run serve
```
